# Felis Catus v0.1
UCI Chess engine based on [Bobcat](https://github.com/Bobcat/bobcat) 8.0 by Gunnar Harms



## Overview

- Complete refactored codebase
- Tuner is separated from engine and support CLI to tune parameters
- C++20
- CMake
